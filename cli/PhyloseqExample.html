Scripts

We will use scripts from the MG-RAST-Tools repository on GitHub (https://github.com/MG-RAST/MG-RAST-Tools). Feel free to copy, edit or contribute.

  • mg-compare-taxa.py
  • mg-mg-compare-normalize.py
  • mg-biom-merge.py
  • mg-biom2metadata
  • mg-biom2taxa

Workflow

We have to create abundance profiles for both sample groups and import them into R to compare them with phyloseq.

  • Create Greengenes abundance profiles for both sample groups ( Amplicon  and 16S from WGS ).
  • mg-compare-taxa.py --ids $WGS --level genus --source Greengenes --format biom --evalue 8 --hit_type all > wgs.greengenes.biom
  • mg-compare-taxa.py --ids $AMPLICON --level genus --source Greengenes --format biom --evalue 8 --hit_type all > amplicon.greengenes.biom
  • We have to combine the two profiles.
  • mg-biom-merge.py wgs.greengenes.biom amplicon.greengenes.biom > merged.greengenes.biom
  • Normalizing the profiles, this can be done in R too.
  • mg-mg-compare-normalize.py --input merged.greengenes.biom --output merged.greengenes.normalized.biom
  • The OTU data from the biom file, metadata and taxonomy information are needed to create a phyloseq object [pyholoseq_object = phyloseq(OTUTABLE, TAX , META)]. For this we extract the metadata and taxonomy information from the biom file.

  • mg-biom2metadata --input amplicon.greengenes.biom > amplicon.metadata
  • mg-biom2taxa --input amplicon.greengenes.biom > amplicon.taxa

    Same for wgs.greengenes.biom and merged.greengenes.biom
  • Switch to R for further analysis.
